# College-Project
This is a college project developed and co-ordinated by 5 college students

## Project Components

### Clothing Website
A complete e-commerce clothing website with multiple pages:
- Home page (index.html)
- Men's collection (men.html)
- Women's collection (women.html)
- About page (about.html)
- Contact page (contact.html)
- Shopping cart (cart.html)

Features: Responsive design, product images, navigation, and interactive elements.

### E_commerce Website Clone (HTML/CSS)
A e-commerce interface built with HTML and CSS:
- Products
- Highlights
- Section Pages
- General styling



## How to View the Clothing Website
1. Clone this repository
2. Open `Clothingwebsite/index.html` in your web browser
